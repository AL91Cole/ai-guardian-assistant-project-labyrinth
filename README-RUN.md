# Quick Run Guide

## Requirements

- Node.js installed
- npm installed

## Run locally

```bash
cd project-labyrinth
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production check

```bash
npm run build
npm start
```

## Notes

- The app uses SQLite and stores data in `data/labyrinth.sqlite`.
- The User Private Vault and Labyrinth are defensive simulations only.
